# ProstateX-Classification

Prostate cancer is the most common cancer in men, and multi-parametric MRI (mpMRI) has become the standard non-invasive imaging method for lesion detection and clinical significance assessment. The PROSTATEx dataset provides 346 patients with annotated mpMRI lesions across T2-weighted (T2W), diffusion-weighted (DWI), and apparent diffusion coefficient (ADC) sequences with binary clinical significance labels, all of which help enable a lesion-level binary classification task. I evaluate a logistic regression baseline on T2W image patches and compare three convolutional neural network (CNN) configurations trained on T2W alone, DWI+ADC, and all three modalities combined, to then conduct a modality ablation study with three independent training runs per model and report both AUC-ROC and Average Precision (AP) to account for class imbalance. The multi-modal CNN achieves the highest mean AUC of 0.740 across three runs, with the three-modality network providing the strongest individual signal, showing that using multiple MRI imaging types together we can capture more information about deciphering if cancer is benign or malignant.
